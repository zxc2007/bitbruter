# Bitbruter
```

# Usage:
### Python3+

```
git clone https://github.com/Py-Project/bitbruter.git

cd bitbruter && pip install -r requirements.txt

python bitbruter.py

```
### Windows

cd dist
start bitbruter.exe
```
